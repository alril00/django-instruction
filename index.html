Создаем систему авторизации на Django: пошаговая инструкция

Почему именно Django?

Django — это мощный и гибкий фреймворк для разработки веб-приложений на Python. Он помогает быстро создавать безопасные и масштабируемые сайты. В нашем случае, благодаря встроенному админ-панели, готовым компонентам аутентификации и многочисленным расширениям, мы сможем реализовать функционал регистрации и входа без лишних усилий.

Что мы собираемся сделать

Перед началом работы важно понять конечную цель: при открытии сайта пользователь сразу попадает на страницу входа. Не зарегистрирован — переходит на страницу регистрации, вводит свои данные, и они сохраняются в базе данных. После успешного входа — попадает в свой профиль, где видит свои личные данные и может выйти из системы. Всё это — безопасно, удобно и быстро.

Цель проекта

При открытии http://127.0.0.1:8000 автоматически загружается страница входа. Если пользователь не зарегистрирован — он может перейти на страницу регистрации, ввести данные, которые сохранятся в базе. После входа откроется личный профиль, где можно увидеть свои данные и выйти.

Создание кастомного пользователя

Стандартная модель пользователя Django — это хорошо, но для наших целей лучше создать свою собственную. Мы делаем это, чтобы использовать только email для входа и иметь возможность расширять профиль по необходимости. Здесь важна правильная настройка менеджера и модели.

В accounts/models.py создаем пользователя:

from django.contrib.auth.models import AbstractUser, BaseUserManager
from django.db import models

class UserManager(BaseUserManager):
    def create_user(self, email, password=None, **extra_fields):
        if not email:
            raise ValueError('Email обязателен')
        email = self.normalize_email(email)
        user = self.model(email=email, **extra_fields)
        user.set_password(password)
        user.save(using=self._db)
        return user

    def create_superuser(self, email, password=None, **extra_fields):
        extra_fields.setdefault('is_staff', True)
        extra_fields.setdefault('is_superuser', True)
        return self.create_user(email, password, **extra_fields)

class User(AbstractUser):
    username = None
    email = models.EmailField(unique=True)
    first_name = models.CharField(max_length=100)
    last_name = models.CharField(max_length=100)

    objects = UserManager()  # ← подключили кастомный менеджер

    USERNAME_FIELD = 'email'
    REQUIRED_FIELDS = ['first_name', 'last_name']

    def __str__(self):
        return self.email

from djoser.serializers import UserSerializer as BaseUserSerializer

auth.js 

function saveToken(token) {
    localStorage.setItem('token', token);
}

function getToken() {
    return localStorage.getItem('token');
}

function removeToken() {
    localStorage.removeItem('token');
}

function logout() {
    removeToken();
    alert('Вы вышли');
    window.location = '/login/';
}

function requireAuth() {
    if (!getToken()) {
        alert('Требуется вход');
        window.location = '/login/';
    }
}

login.js 

document.getElementById('login-form').onsubmit = async (e) => {
    e.preventDefault();
    const email = document.getElementById('email').value;
    const pass = document.getElementById('password').value;
    const btn = e.submitter;
    const msg = document.getElementById('msg');
    btn.disabled = true;
    msg.innerHTML = '';

    const res = await fetch('/api/auth/jwt/create/', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ email, password: pass })
    });

    if (res.ok) {
        const data = await res.json();
        saveToken(data.access);
        window.location = '/profile/';
    } else {
        msg.innerHTML = 'Неверный email или пароль';
        btn.disabled = false;
    }
};

register.js 

document.getElementById('reg-form').onsubmit = async (e) => {
    e.preventDefault();
    const email = document.getElementById('email').value;
    const fname = document.getElementById('fname').value;
    const lname = document.getElementById('lname').value;
    const p1 = document.getElementById('pass1').value;
    const p2 = document.getElementById('pass2').value;
    const msg = document.getElementById('msg');

    if (p1 !== p2) return msg.innerHTML = 'Пароли не совпадают';
    if (p1.length < 8 || !/[a-z]/i.test(p1) || !/\d/.test(p1)) 
        return msg.innerHTML = 'Пароль: ≥8, буквы+цифры';

    const res = await fetch('/api/auth/users/', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            email,
            first_name: fname,   // ← first_name, не firstName!
            last_name: lname,    // ← last_name, не lastName!
            password: p1,
            re_password: p2      // ← обязательно, если USER_CREATE_PASSWORD_RETYPE = True
        })
    });

    if (res.ok) {
        alert('Успешно! Теперь войдите.');
        window.location = '/';
    } else {
        const err = await res.json();
        console.log('Ошибка сервера:', err);  // ← смотри в DevTools → Console!
        msg.innerHTML = err.email ? 'Email занят' : 
                       err.password ? 'Слабый пароль' :
                       err.re_password ? 'Пароли не совпадают' :
                       'Ошибка регистрации';
    }
};

register.js 

requireAuth();

const token = getToken();
fetch('/api/auth/users/me/', {
    headers: { 'Authorization': 'Bearer ' + token }
})
.then(r => {
    if (!r.ok) throw 'Нет доступа';
    return r.json();
})
.then(user => {
    document.getElementById('loading').style.display = 'none';
    document.getElementById('data').style.display = 'block';
    document.getElementById('email').textContent = user.email;
    document.getElementById('fname').textContent = user.first_name;
    document.getElementById('lname').textContent = user.last_name;
    document.getElementById('date').textContent = new Date(user.date_joined).toLocaleString();
})
.catch(() => {
    removeToken();
    document.getElementById('err').textContent = 'Сессия истекла. Войдите снова.';
    document.getElementById('err').style.display = 'block';
    setTimeout(() => window.location = '/login/', 2000);
});
